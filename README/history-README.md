# 福州大学红色校史展示网站 README

# 一、项目概述

本项目是一个福州大学红色校史主题的单页展示网站，通过时间轴、红色打卡点、校园地图交互等模块，系统呈现福州大学自 1958 年建校以来的红色发展脉络与精神传承。网站采用原生前端技术构建，支持响应式布局，兼顾美观性与交互性，适配桌面端、平板及移动端设备。

# 二、功能特点

## 2.1 核心展示功能

- **红色校史时间轴**：按1958 年建校奠基、1978 年改革传承、2008 年校史馆落成、2021 年数字化建设、未来发展展望五大时间节点，分层展示校史内容

- **红色打卡点展示**：以卡片形式呈现校史馆、旗山校区图书馆、革命校友纪念园等实体打卡点的地址、开放时间及特色

- **校园地图交互**：地图标注红色打卡点，支持鼠标悬停显示简介、点击查看详情

- **多媒体预览**：校史图片、打卡点图片支持点击放大查看，搭配图文说明提升展示效果

## 2.2 交互体验优化

- **详情分层展开**：校史详情、打卡点信息默认折叠，点击触发器可展开 / 收起，长文本支持弹窗放大阅读

- **动态视觉效果**：卡片悬浮阴影强化、元素加载渐入动画、导航栏滚动样式变化

- **快捷操作支持**：键盘ESC键一键关闭所有弹窗及展开的详情面板

- **平滑滚动**：页面内锚点跳转采用平滑滚动，提升浏览体验

## 2.3 视觉设计特色

- **主题色调**：以福大红色#c8102e和校徽蓝色#0055a4为主色调，契合红色校史主题

- **纹理与质感**：定制 SVG 淡色背景纹理，增强页面层次感，避免视觉单调

- **响应式适配**：针对不同设备尺寸优化布局，移动端自动调整时间轴为单列展示、导航栏堆叠排列

# 三、技术栈

- **HTML5**：语义化标签构建页面结构，确保可读性与可维护性

- **CSS3**：自定义 CSS 变量统一管理样式、Flex/Grid 布局实现响应式排版、CSS 动画实现交互反馈

- **原生 JavaScript**：无框架依赖，实现弹窗控制、详情展开、地图交互等功能

- **Font Awesome**：图标库丰富页面视觉元素

- **SVG**：自定义背景纹理，轻量化且不失质感

# 五、页面结构说明

- **导航栏**：固定顶部导航，包含首页、红色校史、打卡点、数字展馆、关于我们五个导航项，当前页面自动高亮

- **标题区域**：居中展示页面主标题，下方红色装饰线强化视觉焦点

- **时间轴区域**：核心内容区，按时间顺序排列校史节点，左右交替布局（大屏）/ 单列布局（小屏）

- **打卡点区域**：红色背景区块，网格布局展示打卡点卡片，支持详情展开

- **校园地图区域**：嵌入校园地图，标注打卡点位置，支持交互提示

- **弹窗组件**：包含图片预览弹窗、文本详情弹窗，统一管理弹窗状态

- **页脚**：展示网站版权信息与学校办学宗旨

# 六、部署与使用

## 6.1 部署方式

将项目文件完整上传至 Web 服务器根目录，确保assets目录路径正确

## 6.2 访问方式

直接访问服务器域名 / IP 地址，或打开index.html文件本地预览

## 6.3 交互操作

- 点击图片可放大查看原图及说明

- 点击 “查看详细校史 / 打卡详情” 可展开对应内容

- 长文本内容点击 “放大查看完整内容” 可弹窗阅读

- 地图点位鼠标悬停显示简介，点击查看详情

- 按ESC键可快速关闭所有弹窗和展开的内容

# 七、自定义指南

## 7.1 内容更新

- 校史节点：复制现有timeline-item结构，修改年份、标题、图片路径及详情文本

- 打卡点：新增checkin-card卡片，配置图片、名称、描述及详情信息

- 地图点位：添加map-point元素，设置坐标、关联数据及样式

## 7.2 样式调整

修改 CSS 根变量--fd-red/--fd-blue等，可快速替换主题色

## 7.3 功能扩展

可结合后端接口实现校史内容动态加载，或添加留言、分享等功能

# 八、浏览器兼容

- 兼容现代浏览器：Chrome、Firefox、Safari、Edge 最新版本

- 支持 IE11 及以上版本（需调整部分 CSS3 属性）
